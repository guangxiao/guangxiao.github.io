Vue 源码分析（一）：vue-flow

作者：陈广宵
日期：2018-05-09
分类：JavaScript

Flow 是 facebook 出品的 JavaScript 静态类型检查工具，Vue 的源码采用了 Flow 做静态类型检查，所以要熟悉 Vue 的源码分析，首先要了解 Flow。

Flow 有两种类型检查

  • 类型推断：通过变量的使用上下文来推断出变量类型，然后根据这些推断来检查类型。
  • 事先注释好我们期待的类型，Flow 会基于这些注释来判断。
